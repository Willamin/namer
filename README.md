# namer

A quick shard to provide random, alliterative names based on a preset list of adjectives and nouns.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     namer:
       github: Willamin/namer
   ```

2. Run `shards install`
